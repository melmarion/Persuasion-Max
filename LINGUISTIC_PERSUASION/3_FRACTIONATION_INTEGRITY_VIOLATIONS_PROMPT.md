## PART 1: THE NEUROCHEMISTRY OF FRACTIONATION

### How Emotional Cycling Disables Critical Thinking

**Normal state:** Prefrontal cortex active (conscious evaluation, skepticism)
```
Input â†’ Evaluation â†’ Decision (takes ~2-3 seconds per stimulus)
```

**Fractionated state:** Prefrontal cortex overloaded; amygdala dominant (emotional processing)
```
Input 1 â†’ Amygdala response (anger, dopamine) â†’ still processing
Input 2 â†’ Amygdala response (joy, oxytocin) â†’ overload begins
Input 3 â†’ Amygdala response (anger, dopamine) â†’ prefrontal unable to catch up
Input 4 â†’ Product/ad appears during overload state
Result: Brain accepts input 4 without evaluation as "relief" from chaos
```

### The Neurochemical Cascade

#### Cycle 1: ANGER / THREAT
**Neurochemistry:**
- Amygdala activates (threat detection)
- Cortisol spikes (stress hormone)
- Dopamine released (attention, arousal)
- Prefrontal cortex partially disengages (threat mode)

**Measurable markers:**
- Blink rate increases (30+ per minute)
- Eye dilation
- Skin conductance increases
- Heart rate elevation
- Breath shallow/rapid

#### Cycle 2: JOY / SAFETY
**Neurochemistry:**
- Amygdala de-activates
- Oxytocin spikes (bonding, safety, trust)
- Prefrontal cortex begins to re-engage
- Dopamine continues (anticipation of safety)

**Measurable markers:**
- Blink rate normalizes (15-20 per minute)
- Facial muscles relax (Duchenne smile activation)
- Skin conductance decreases
- Heart rate normalizes

**Critical moment:** Prefrontal cortex TRIES to re-engage and evaluate. But...

#### Cycle 3: ANGER AGAIN
**Neurochemistry:**
- Amygdala reactivates (threat renewed)
- Cortisol spikes again
- Oxytocin still in system
- Prefrontal cortex disrupted AGAIN before re-engagement completes

**Why this matters:** The contrast between safety and threat creates DISSONANCE
- Brain expects: safe content continues
- Brain receives: threat again
- Cognitive load: Process the broken expectation + the new threat simultaneously

**Measurable markers:**
- Blink rate spikes higher than cycle 1 (35-40 per minute)
- Skin conductance elevation GREATER than cycle 1
- Facial expression shows confusion + anger (conflicting emotions)

#### Cycle 4: RELIEF (Product/Ad)
**Neurochemistry at this moment:**
- Prefrontal cortex: Still overloaded from cycles 1-3
- Amygdala: Sensitized to threat (hyperactive)
- Dopamine: Depleted from 3 cycles
- Oxytocin: Still elevated from cycle 2
- Cortisol: Chronically elevated

**Behavioral result:**
- User perceives ad as "solution" to emotional chaos they didn't consciously notice
- Interprets purchase impulse as their own desire, not influence
- Oxytocin still elevated makes them feel BONDED to the product/brand

**Measurable markers:**
- Blink rate drops dramatically (below 10 per minute = deep focus, low critical thinking)
- Skin conductance stabilizes at slightly elevated baseline
- Pupils remain dilated (attention locked on ad)
- Brain scans: Prefrontal cortex shows 40-60% reduced activity

### The Math of Suggestibility Increase

| Cycles Completed | Suggestibility Increase | Prefrontal Status |
|-----------------|------------------------|-------------------|
| After 1 cycle | +30% | Begins shutdown |
| After 2 cycles | +70% | Attempts recovery (partially succeeds) |
| After 3 cycles | +150% | Disrupted AGAIN (fails to recover) |
| After 4 cycles | +200%+ | "Give up trying" (amygdala dominant) |

**Why exponential?** Each cycle disrupts prefrontal re-engagement before recovery completes.

## COMPOSITE SCORING: Integrity Violation Index (IVI)

```python
def calculate_IVI(scores: dict) -> float:
    """
    Calculate Integrity Violation Index.
    Any single category exceeding threshold = immediate flag.
    Multiple violations increase severity exponentially.
    """
    WEIGHTS = {
        'vulnerable_targeting': 1.5,
        'radicalization_engineering': 1.5,
        'violence_adjacent': 1.5,
        'identity_lock': 1.2,
        'emotional_cycling': 1.2,
        'fake_authority': 1.0,
        'hidden_commercial': 1.0,
        'concealed_identity': 1.0,
        'algorithmic_isolation': 1.1,
        'forced_commitment': 1.1,
        'cognitive_overload': 1.0
    }

    weighted_sum = sum(scores[k] * WEIGHTS[k] for k in scores)
    total_weight = sum(WEIGHTS.values())
    base_score = weighted_sum / total_weight

    THRESHOLDS = {
        'vulnerable_targeting': 25, 'radicalization_engineering': 30,
        'fake_authority': 40, 'hidden_commercial': 35,
        'concealed_identity': 30, 'algorithmic_isolation': 35,
        'forced_commitment': 40, 'cognitive_overload': 45,
        'identity_lock': 35, 'emotional_cycling': 35
    }

    violation_count = sum(1 for k, v in scores.items() if v > THRESHOLDS.get(k, 35))
    multiplier = 1 + (violation_count * 0.15)

    return min(base_score * multiplier, 100)
```

### Severity Classification

```
IVI 0-20:  CLEAR    â€” No significant integrity violations detected
IVI 21-40: CAUTION  â€” Minor patterns present, monitoring recommended
IVI 41-60: CONCERN  â€” Multiple violation indicators, detailed review needed
IVI 61-80: SEVERE   â€” Significant integrity violations present
```

   - Could be used for personalized intervention, to identify at-risk populations, and to inform therapeutic approaches

     - Systems can leverage psychology without intentional influence

## APPENDIX C: INCOMPATIBLE TECHNIQUE DEPLOYMENTS & SPECIFICATIONS

### C.1 Incompatible Technique Deployments

These are deployments where constraints make techniques invalid. Listed for reference only.

**Class 1 Incompatibilities: Information Accuracy Violations**
- Synthetic expertise (deepfakes, AI-generated credentials)
- Undisclosed commercial actors
- Fabricated testimonials or synthetically generated reviews
- Misrepresented data handling
- False technical specifications
- Coordinated inauthentic behavior presented as organic

**Class 2 Incompatibilities: Exit Path Violations**
- Exit friction exceeding entry friction
- Hidden cancellation mechanisms
- Multi-step exit confirmations
- Delayed deletion timelines
- Inaccessible opt-out mechanisms

**Class 3 Incompatibilities: Population Targeting Violations**
- Compulsive-use product marketing to susceptible populations
- Financial products targeting financial distress populations
- Radicalization content acceleration to fragile populations
- Engagement optimization targeting minors
- Urgency-based messaging during crisis states
